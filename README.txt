1) clone or form project
2) 'cd' into the project directory
3) 'virtualenv venv' in terminal to create virtual environment
4) 'source venv/bin/activate' to start virtual environment
5) 'pip install -r requirements.txt' to install nesseccary python packages
6) In settings.py, add a SECRET_KEY on line 24.
7) 'python manage.py runserver' to run on localhost
8) http://127.0.0.1:8000/
